- UI package manager is yarn (use `yarn add` / `yarn add -D`, run scripts with `yarn <script>`)

- SCSS files live in ui/src/styles/ with subdirectories: tokens/, mixins/, base/, utilities/
- All color hex values must live exclusively in _colors.scss — no hard-coded hex values elsewhere
