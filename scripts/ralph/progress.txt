## Codebase Patterns
- Use static properties object approach for Lit components (no decorators) — tsconfig has useDefineForClassFields: true which conflicts with Lit decorators
- Always add HTMLElementTagNameMap declaration in Lit component files for TypeScript awareness
- All Lit components registered via customElements.define(), imported by side-effect in ui/src/components/lit/index.ts
