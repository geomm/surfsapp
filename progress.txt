## Codebase Patterns
- UI typecheck: run `yarn typecheck` from ui/ (no root package.json). It runs `vue-tsc --noEmit`.
- Lit components registered via `./components/lit/index` import in ui/src/main.ts.
- Pinia uses Options API `defineStore` style for this project.
- IndexedDB via Dexie: `ui/src/db.ts` exports `db` (SurfsAppDB) with `favourites` (&beachId) and `settings` (&key) tables. Import as `import { db } from '../db'` from stores/components.
- New lucide icons: import in `ui/src/components/lit/surf-icon.ts`, add to ICONS map keyed by kebab-case name. Lucide exports `Map` and `List` as PascalCase.
- Route names are referenced via `router.push({ name: 'xxx' })`. Existing names: `beach-detail`. Phase 13 adds `beach-map`.
- FAB / overlay pattern: position fixed, use env(safe-area-inset-*) on offsets, `--color-ocean-800` is the brand accent.
- MapLibre: import `'maplibre-gl/dist/maplibre-gl.css'` in any view that mounts a map (BeachMap.vue does NOT â€” it relies on the importer to provide CSS). Without it controls/markers render unstyled.
- Beach coords on `Beach` type are `{ lat, lon }`; MapLibre expects `[lon, lat]` order â€” easy to flip by mistake.
- MapLibre paint expressions cannot consume CSS custom properties â€” read them via `getComputedStyle(document.documentElement).getPropertyValue('--color-...')` at map init and pass the resolved hex into a `match` expression on a feature property.
- Demotiles style glyphs include `Noto Sans Regular` â€” use it as the `text-font` value for symbol layers there. Other demotiles fonts are not guaranteed.
- maplibre-gl v5 `GeoJSONSource.getClusterExpansionZoom(clusterId)` returns a Promise<number> (not a callback). Cast `map.getSource(id)` to `maplibregl.GeoJSONSource` to access cluster helpers.
- Cluster + non-cluster filters: `['has', 'point_count']` for clusters, `['!', ['has', 'point_count']]` for individual features. Add layers in render order â€” clusters and individual circles are mutually exclusive via the filters, so order matters only for stacking with future overlays.
